# DÓNDE COMPRO.COM 📍
**App de localización y gestión de productos en comercios cercanos.**

Este sitio permite a los usuarios encontrar productos en tiendas locales mediante geolocalización.

## 🚀 Estado del Proyecto
- [x] Definición de estructura.
- [ ] Creación de Base de Datos (PostgreSQL).
- [ ] Diseño de Interfaz (React).
- [ ] Panel de Comerciantes y Administración.

## 🛠️ Especificaciones
- Capacidad: 100,000 comercios y 80,000 productos.
- Filtros: Producto, rubro, precio, ubicación y horario.
